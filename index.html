Projects

Here, you'll find a collection of projects that bring my skills to life, each one highlighting a unique aspect of what I can do.

AirBRB

This project is a clone of AirBnB. This project used JavaScript with Express.js for the backend server and React and Redux for the frontend. Sequelize was used as the ORM and the data is stored using a ProstgreSQL database.
Github Live Link

NBA-Rank

NBA Rank is a player-based NBA site where you can view the top 5 players in various stat categories over the past 5 NBA seasons. You can make posts on player pages, like players, and add players to your top 5! This project implements a RESTful API for retrieving player data to ensure efficient and scalable data retrieval and management.
Github Live Link

HoneyStock

Robinhood clone project. This was a group project built in collaboration with classmates at App Academy. This project uses Python with Flask as the backend server with React and Redux for the frontend. SQLAlchemy was used for the ORM with a ProstgreSQL database. The stock data is retrieved from the polygon.io API.
Github Live Link

About

As a passionate software engineer with several years of professional experience and a lifelong enthusiasm for gaming, I am eager to transition into a role where I can merge my technical expertise with my love for sports and interactive experiences. My journey has been shaped by self-directed learning and a comprehensive Boot Camp at App Academy, where I completed numerous projects utilizing a diverse tech stack, including JavaScript, Python, React/Redux, Express, Sequelize, SQLAlchemy, and Flask.

This foundation has enabled me to develop dynamic web applications, optimize processes, and contribute to collaborative projects that emphasize my commitment to high-quality software development. My passion for gaming has not only fueled my creativity but also sharpened my problem-solving abilities and deepened my understanding of user engagement and interactive design. I thrive in fast-paced environments where innovation meets technical challenges.

I’m an analytically minded digital professional with a growing focus on SEO and a passion for understanding how people search, discover, and engage with content online. I thrive in roles that blend data, structure, and problem‑solving — whether that’s optimizing on‑page elements, analyzing performance metrics, or improving workflows to drive better outcomes.

My experience has been built on accuracy, consistency, and systems thinking. I’ve managed complex operational processes, maintained high‑stakes data integrity, collaborated across departments, and supported teams through fast‑changing requirements. Those same strengths now shape the way I approach SEO: with attention to detail, curiosity, and a commitment to continuous improvement.

I’m comfortable working with tools like Google Analytics, Google Search Console, Ahrefs, Sistrix, and Screaming Frog, and I’m actively expanding my skills in keyword research, on‑page optimization, internal linking, and performance analysis. I enjoy breaking down problems, spotting patterns, and turning insights into clear, actionable steps.

I’m especially drawn to SEO in the sports and betting space — the pace, the competition, and the data‑driven nature of the industry align perfectly with how I think and work. I’m excited by opportunities where I can contribute to organic growth, support international SEO efforts, and collaborate with teams to improve rankings, content quality, and user experience.

Always learning. Always refining. Always looking for ways to make systems — and results — better.
